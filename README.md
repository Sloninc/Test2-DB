# Test2
Тестовое задание №2 "Работа с базой данных"
Создать приложение для ведения базы данных результатов проверок оборудования с двумя таблицами:

Проверки (Tests)

Наименование поля			Тип данных			Описание данных					Обязательное поле		Ключевое поле
TestId						Int					Идентификатор проверки			Да						Да
TestDate					Smalldatetime		Дата проведения проверки		Да						Нет
BlockName					Nvarchar(50)		Наименование проверяемого блока	Да						Нет
Note						Nvarchar(200)		Примечание						Нет						Нет

Параметры (Parameters)

Наименование поля			Тип данных			Описание данных					Обязательное поле		Ключевое поле
ParameterId					Int					Идентификатор параметра			Да						Да
TestId						Int					Идентификатор проверки			Да						Да
ParameterName				Nvarchar(200)		Наименование параметра			Да						Нет
RequiredValue				Decimal				Требуемое значение параметра	Да						Нет
MeasuredValue				Decimal				Измеренное значение параметра	Да						Нет

Требования к приложению
База данных должна быть создана с помощью любой СУБД (желательно MS SQL Server), 
приложение – разработано в среде MS Visual Studio на языке С#, и представлять собой WPF с полями для просмотра, 
ввода и модификации данных в базе данных, причем параметры блока должны отображаться в таблице. 
Для проверки предоставляются исходный код приложения и инсталляционного пакета, инсталляционный пакет приложения, 
скрипт на T-SQL для создания базы данных и заполнения таблиц тестовыми данными, указания по подключению к базе данных.
